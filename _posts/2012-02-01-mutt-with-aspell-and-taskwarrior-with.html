Mutt with aspell and Taskwarrior with VITtk

恥ずかしい英語のスペルミスを減らす！

Prepare necessary programs

aptitude install vim mutt aspell urlview task

Setting Mutt

add MUTT to LXDE menu
[Desktop Entry]  
 Encoding=UTF-8  
 Exec=lxterminal --command "mutt -F /home/tknv/.muttrc"  
 Icon=/home/tknv/Pictures/icon/mutt-mailer.jpg  
 Type=Application  
 Terminal=true  
 Name=mutt  
 GenericName=mutt   
 StartupNotify=false   
 Categories=Network
*There is no mutt icon,I used 
setting mailcap and mime type
$ cp /etc/mailcap ~/.mailcap_path  
 $ cp /etc/mime.types ~/.mime.types

Setting IMAP

for gmail + gpg
.muttrc
set imap_user = "foo@gmail.com"  
  set smtp_url = "smtp://foo@smtp.gmail.com:587/"  
  set from = "foo@gmail.com"  
  set realname = "foo woo"  
  set folder = "imaps://imap.gmail.com:993"  
  set spoolfile = "+INBOX"  
  set postponed="+[Google Mail]/Drafts"  
  set signature="~/.signature"
set quit  

  set header_cache=~/.mutt/cache/headers  
  set message_cachedir=~/.mutt/cache/bodies  
  set certificate_file=~/.mutt/certificates  

  set sort = threads  
  set sort_aux = reverse-last-date-received  
  set sort_re  
  set mailcap_path=~/.mailcap  

  # allow mutt to open new imap connection automatically  
  # unset imap_passive  
  # keep imap connection alive by polling intermittently (time in seconds)  
  set imap_keepalive = 300  
  # how often to check for new mail (time in seconds)  
  set mail_check = 120  
  set timeout = 300  
  # Display download progress every 5K  
  set net_inc=5  

  bind editor  noop  
  macro index in "=INBOX" "Go to inbox"  
  macro index al "=[Google Mail]/All Mail" "Go to all mail"  
  macro index se "=[Google Mail]/Sent Mail" "Go to Sent Mail"  
  macro index im "=[Google Mail]/Important" "Go to Important Mail"  
  macro index ho "=Hold" "Go to Hold Mail"  
  macro index jo "=Job" "Go to Job Mail"  
  macro index ac "=Action" "Go to Action Mail"  
  macro index re "=Respond" "Go to Respond Mail"  
  macro index wa "=Waiting" "Go to Waiting Mail"  
  macro index df "=[Google Mail]Drafts" "Go to drafts"  
  macro index,pager SP "=[Google Mail]/Spam" "mark message as spam"  
  macro index,pager HA "=INBOX" "mark message as ham"  
  macro index,pager HO "=Hold" "mark message as Hold"  
  macro index,pager AC "=Action" "mark message as Action"  
  macro index,pager RE "=Respond" "mark message as Respond"  
  macro index,pager WA "=Waiting" "mark message as Waiting"  
  macro index,pager SA "=Save" "mark message as Save"  

  set move = no  #Stop asking to "move read messages to mbox"!  

  alternative_order text/plain text/html  
  auto_view text/html  
  macro pager \cb 'urlview' 'Follow links with urlview'  

  # GPG  
  source /usr/share/doc/mutt/examples/gpg.rc

Setting VIM

$ vim .bashrc  
$ :autocmd FileType mail :nmap  :w:!aspell -e -c %:e
*Hit f8 key in vim command mode in mutt editing mail, Aspell find spell miss! Here nice muttutor site.

VITtk info dl source

build and install
$ cd   
$ ./configure  
$ make install  
$ mkdir /tmp/.task-test  
$ task rc.data.location=/tmp/.task-test add This is a task  
$ task rc.data.location=/tmp/.task-test add This is another task  
$ vittk -h  
$ vittk -ds /tmp/.task-test -d > /tmp/vittk-debug.log  
$ more /tmp/vittk-$USER.log  
$ more /tmp/vittk-debug.log
add VITtk to LXDE menu
$ vim .local/share/applications/vittk.desktop
