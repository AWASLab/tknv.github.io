setup lubuntu11.04 for IBM X31
for developing android and Japanese env. too


install lubuntu by usb
partitioning disk
/dev/sda1 20GB mount point /
for system
/dev/sda3 all others mount point /home
for data
swap 4000MB
setup repository
Open Menu –> Systeme Tools –> Synaptic Package Manager,Manager –> Settings –> Repositories, check the Canonical Partners.
add APT repo,
deb http://android-notifier.googlecode.com/svn/repos/deb-repo stable main

[Desktop Entry]  
Encoding=UTF-8

Icon=/eclipse/icon.xpm  
Type=Application  
Terminal=false  
Name=eclipse  
GenericName=eclipse  
StartupNotify=false

Android SDK
google instruction is best:see

set build android environment
get build tool etc – CM instruction is best:see

sudo apt-get install git-core gnupg flex bison gperf libsdl1.2-dev libesd0-dev libwxgtk2.6-dev squashfs-tools build-essential zip curl libncurses5-dev zlib1g-dev sun-java6-jdk pngcrush schedtool

get source

mkdir ~/bin  
curl http://git-repo.googlecode.com/files/repo-1.12 > ~/bin/repo  
chmod a+x ~/bin/repo  
mkdir android/foo  
cd android/foo  
repo init -u git://android.git.kernel.org/platform/manifest.git  
repo sync  
###### It will take 2hours more. ######

build source

. build/envsetup.sh
