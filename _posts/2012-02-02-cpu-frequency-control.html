Linux CPUクロックを操作する。
FluxBox: add CPU frequency control in menu
# vim .fluxbox/menu


CPU frequency control by script for Archlinux.
command below it at terminal.
# pacman -S cpufrequtils
# vim /etc/rc.conf
...
MODULES=(acpi-cpufreq ...)
...
# sudo rc.d start acpi-cpufreq

Checking.
# cpufreq-info

It should return CPU info.
then.
# vim ~/Scripts/CPU-performance

# chmod +x ~/Scripts/CPU-performance
# ./home/foo/Scripts/CPU-performance
Now machine foo's CPU is Full GHZ clock!
