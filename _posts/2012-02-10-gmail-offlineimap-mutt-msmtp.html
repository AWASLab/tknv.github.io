I felt strange, when I knew off line imap. It is same like a pop3.
But not.オフラインimapを聞いたとき、pop3と何が違うのと思ったけど、全然違うわ

install software

debian kind.
  aptitude install offlineimap mutt msmtp
archlinux.
  pacman -S offlineimap mutt msmtp
config offlineimap
--.offlineimaprc--

This configure will not sync All Mail and Bin folders.
config mutt
--.muttrc--

config msmtp
--.msmtprc--

using

offlineimap
It will take long, also first run will not make ~/.mutt/mailboxes or need to wait until sync finish. anyway run offlineimap 2 times then,
mutt
