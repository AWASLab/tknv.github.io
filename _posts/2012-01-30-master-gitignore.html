gitignore

To add deep directory file only and ignore other files.

ex. want to add need.cxx only.

There is
/hoo/no-need.cxx
/hoo/moo/no-no-need.cxx
/hoo/moo/foo/need.cxx
make .gitignore file at each directory.
/hoo/.gitignore
*    
!moo
/hoo/moo/.gitignore
*    
!foo
/hoo/moo/foo/.gitignore
!*.cxx
“!*.cxx” isn’t smart, But I don’t know another way now.
~$>git add -A
